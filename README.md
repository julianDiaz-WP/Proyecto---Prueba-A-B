# Proyecto---Prueba-A-B
Este repositorio corresponde al proyecto sobre el desarrollo y análisis de la prueba A/B

El entorno utilizado para el desarrollo de este proyeto fue "Python 3.13.5" debido a que este contaba con las librerías necesarias para el desarrollo de este proyecto. 
El archivo "Prueba AB" es el que contiene el desarrollo completo del pryecto donde se analizó le impacto de los cambios puestos a prueba en la aplicación de venta de productos (caso ficticio)
